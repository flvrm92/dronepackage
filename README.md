## About the challenge

It was used the [greedy algorithm](https://www.geeksforgeeks.org/greedy-algorithms/) to reach out the solution.
The project also has a service to validate the input data before starting the process.

## How to run

To run the project, you need to have .net core 7.0 installed on your machine.
You must set the “InputFile” path to the appsettings.json file (you can find an Input model example in the Solution Items folder).

## Tools used

Used tools for this project include .net CLI and visual studio 2022

## Libraries and dependencies

The only dependency used was the [Microsoft.Extensions.Hosting](https://learn.microsoft.com/en-us/dotnet/core/extensions/generic-host) to configure a generic host for the application.
